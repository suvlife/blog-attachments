# 四象限矩阵模板

基本面 × 技术面双维度诊断，用于批量股票比较。

## 四象限定义

| 象限 | 基本面 | 技术面 | 定义 | 建议动作 |
|------|--------|--------|------|---------|
| 第一象限 | 好 | 好 | 最佳买点 | 重仓/加仓 |
| 第二象限 | 好 | 跌 | 等待时机 | 持有/逢低加仓 |
| 第三象限 | 差 | 涨 | 警惕泡沫 | 减仓/严格止损 |
| 第四象限 | 差 | 跌 | 坚决清仓 | 清仓/回避 |

## 基本面评分标准（四大师框架）

综合以下维度打分（1-5分）：
- 生意质量（权重25%）：差异化、定价权、粘性、规模、资本效率
- 护城河（权重20%）：品牌、转换成本、网络效应、规模、技术
- 管理层（权重20%）：诚信、战略、执行、资本配置
- 估值（权重20%）：PE/PB历史分位、安全边际、股息率
- 风险（权重10%）：竞争、监管、宏观、财务
- 趋势（权重5%）：行业生命周期位置

**基本面对应象限**：
- 综合评分 ≥ 3.5 → 基本面"好"
- 综合评分 < 3.5 → 基本面"差"

## 技术面评分标准（威科夫方法）

结构完整性评分（0-30分）：
- CO行为逻辑（25%）：意图清晰可解释
- 宏观结构一致性（20%）：大周期支持
- 成交量特征（20%）：符合阶段预期
- 支撑/阻力测试（15%）：关键位多次测试
- 价差特征（10%）：符合预期
- 相对强弱（10%）：vs大盘

**技术面对应象限**：
- 结构评分 ≥ 20 → 技术面"好"
- 结构评分 < 20 → 技术面"差"

## 输出格式

```
┌─────────────────────────────────────────────┐
│ 技术面好 ↑                                  │
│          │  第一象限      │  第三象限        │
│          │  最佳买点      │  警惕泡沫        │
│          │               │                 │
│ ─────────┼───────────────┼──────────────→  │
│          │  第二象限      │  第四象限        │
│          │  等待时机      │  坚决清仓        │
│ 技术面差 ↓               │                 │
│          基本面差          基本面好           │
└─────────────────────────────────────────────┘

| 股票 | 象限 | 基本面评分 | 技术面评分 | 建议 |
|------|------|-----------|-----------|------|
| ...  | ...  | ...       | ...       | ...  |
```
