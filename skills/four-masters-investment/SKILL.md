# 四大师价值投资框架

基于巴菲特、芒格、段永平、李录四位价值投资大师的方法论，提供系统化的投资研究框架。

**版本**: v2.2
**优化方向**: 增加动态估值模块、情绪纪律检查、A股深度覆盖、行业豁免矩阵、概率加权决策、数据来源分级、不适用场景声明

**参考文件**:
- `references/a-share-data-sources.md` — A股实时数据源（东方财富Push API等）
- `references/us-stock-data-sources.md` — 美股数据源（Yahoo Finance v8 chart API等）
- `references/quadrant-matrix-template.md` — 四象限矩阵模板（基本面×技术面双维度批量诊断）

**局限性声明**: 本框架基于历史数据和统计规律，无法预测黑天鹅事件；需要分析者具备基础财务分析能力；不能替代独立判断；过往表现不代表未来收益；任何投资分析都有错误概率。

## 核心哲学

> "买股票就是买公司，买公司就是买公司的未来现金流折现。" —— 段永平

**四大师核心维度**（非严格分工，而是互补视角）：
- **生意本质与商业模式**：段永平强调最多（"好生意"标准），但四大师均重视
- **护城河与竞争优势**：巴菲特提出概念（"经济护城河"），四大师均应用
- **逆向思考与风险识别**：芒格方法论特色（"反过来想，总是反过来想"）
- **长期趋势与文明演进**：李录框架特色（"文明演进框架"）
- **估值与安全边际**：巴菲特核心原则（"以合理价格买入"），四大师均强调

**注意**：四大师的方法论有大量重叠，以上维度划分是为了分析便利，而非严格的角色分工。巴菲特同样极其重视商业模式（"以合理价格买入伟大公司"），芒格同样重视护城河（"护城河是竞争优势的持久性"），李录同样重视估值（"安全边际是价值投资的基石"）。

## 一、投资研究：四大师综合分析

### 执行流程（七模块顺序）

#### 前置：AI研究偏见自觉

**信息丰富度评级**：

| 等级 | 特征 | AI研究陷阱 | 应对策略 |
|------|------|-----------|---------|
| A级（信息充裕） | 上市多年、券商覆盖多 | 共识过强，AI输出趋同于市场定价 | 重点做反面检验：聪明人为什么不买？ |
| B级（信息适中） | 上市1-3年、覆盖有限 | AI可能用"合理推测"填补空白，虚假确定性 | 每个推算数据标注置信度 |
| C级（信息稀缺） | 刚上市/冷门股/新兴市场 | AI会因资料不足而过度保守 | 用第一性原理提问 |

**C级公司的第一性原理研究法**：
1. 客户是谁？为什么付钱？有没有替代选择？
2. 复购靠什么驱动？是习惯、锁定、还是持续创造新价值？
3. 竞争对手拿100亿能复制这门生意吗？
4. 管理层做过什么关键决策？反映了什么判断力和价值观？

**偏见自查清单**：
- [ ] 我的"确定性"感受是来自生意本质，还是来自资料数量？
- [ ] 如果把这家公司的资料量减少一半，我的结论会变吗？
- [ ] AI输出的分析是否与市场共识高度雷同？
- [ ] 是否存在"公开资料很少但生意本质极好"的可能性被低估了？

#### 模块一：数据收集与交叉验证

**必须收集的数据**：
1. 收入结构：最近财年及最近4个季度的分部收入、增速、毛利率
2. 财务指标：近5年收入、净利润、毛利率、经营利润率、自由现金流、现金储备
3. 竞争格局：市场份额、主要竞争对手对比
4. 商业模式与护城河：核心竞争优势来源
5. 管理层：创始人/CEO履历、持股比例、关键决策记录
6. 行业前景：TAM（总可寻址市场）、增长预测
7. 当前估值：市值、PE、PS、PEG、EV/Revenue
8. 多空双方核心论点

**数据交叉验证规则**：
- 每个关键数据点至少2个独立来源
- 发现来源间有差异时，优先采用公司年报/交易所数据
- 所有涉及计算的数据必须精确验算，禁止心算
- 常见错误防范：市值单位（港币亿 vs 人民币亿 vs 美元亿）、FCF口径差异、债务口径、AB股持股比例

**新增：数据时效性分级与来源优先级**

| 数据类型 | 时效要求 | 来源优先级 |
|---------|---------|-----------|
| 股价/市值 | 实时（T+0） | 交易所>Yahoo Finance>券商 |
| 季度财报 | 披露后48小时内 | 公司IR页面>交易所>第三方平台 |
| 年度财报 | 披露后1周内 | 同上 |
| 行业数据 | 季度更新 | 行业协会>券商研报>咨询公司 |
| 管理层动态 | 实时 | 公司公告>监管披露>媒体报道 |

**数据来源分级（必须交叉验证）**：

一级来源（最可靠）：
1. 公司年报/季报（SEC 10-K/10-Q、港交所披露、A股交易所公告）
2. 公司投资者关系页面（IR website）
| 交易所官方数据

**美股数据获取注意（v2.3新增）**：
- Yahoo Finance v7 quote API 和 v10 quoteSummary 已限制免费访问
- 只有 v8 chart API 可用（获取价格/K线/成交量，但不含基本面指标）
- PE/PB/ROE/市值等基本面数据需通过 web_search 或 browser 工具从页面获取
- 详见 `references/us-stock-data-sources.md`

二级来源（较可靠，需交叉验证）：
4. 彭博/路透终端数据
5. 券商研报（注意利益冲突）
6. 财经数据平台（Wind、Choice、FactSet）

三级来源（参考，不单独使用）：
7. 财经媒体（财新、WSJ、FT）
8. 社交媒体/论坛
9. AI生成内容（必须人工验证）

**货币单位校验规则**：
- 港股：注意港币(HKD) vs 人民币(CNY) vs 美元(USD)
- 中概股：注意ADS与普通股比例
- A股：注意总股本 vs 流通股本
- 市值手算校验：股价 × 总股本，与报告市值对比

**A股实时数据快速获取**（详见 `references/a-share-data-sources.md`）：

- 用法：`GET https://push2.eastmoney.com/api/qt/ulist.np/get?fltt=2&fields=f2,f3,f9,f12,f14,f20,f23&secids=1.600011,0.000400`
- 市场代码：`1.xxxxxx`=沪市，`0.xxxxxx`=深市

- 用法：`GET https://query1.finance.yahoo.com/v8/finance/chart/{TICKER}?range=6mo&interval=1d`
- ⚠️ Yahoo v7/v10 API 已不可用（需认证），只用 v8 chart API
- 基本面数据（PE/PB/ROE）需通过 web_search 或 browser 从 Yahoo Finance 页面获取

#### 模块二：生意本质分析 — 段永平"对的生意"

**核心问题**：用一句话定义这门生意的本质

**分析要点**：
- 收入结构拆解（图表）
- 5年盈利能力趋势
- 商业模式画布：一次性销售 vs 订阅/复购？硬件 vs 软件 vs 平台？
- 生态粘性/客户锁定强度
- 毛利率水平与同行对比
- 经营杠杆分析

**段永平式追问**：这门生意好在哪？如果只能用一句话描述，是什么？

**收入漏斗分析（必须执行）**：
画出从"总业务量"到"自由现金流"的完整漏斗，逐层标注金额、比率和具体去向。

```
[顶层业务量指标]（TPV/GMV/订阅用户数×ARPU/产能×售价）
    │ × 抽成率/货币化率/收入转化率
[营收]
    │ - 直接成本（逐项列明：付给谁、大致金额）
[毛利 / 交易利润额]
    │ - 运营费用（研发/销售/管理，逐项列明）
[营业利润]
    │ - 税+利息
[净利润]
    │ + 非现金项目 - 资本支出
[自由现金流]
```

**新增：生意质量评分卡（权重可根据行业特性调整）**

| 维度 | 默认权重 | 评估标准 | 评分(1-5) |
|------|---------|---------|----------|
| 差异化程度 | 25% | 产品/服务是否可被轻易替代 | |
| 定价权 | 25% | 能否在不损失销量的情况下提价 | |
| 客户粘性 | 20% | 复购率/留存率/转换成本 | |
| 规模效应 | 15% | 规模扩大是否带来成本优势 | |
| 资本效率 | 15% | 维持增长所需的资本投入 | |
| **综合生意质量** | 100% | | |

**行业权重调整建议**：
- 平台/网络型公司：客户粘性↑30%、网络效应↑20%（替换"差异化程度"）
- 重资产/制造业：规模效应↑25%、资本效率↑20%
- 科技/专利驱动：差异化程度↑30%、技术壁垒↑20%（替换"客户粘性"）
- 消费品/品牌：定价权↑30%、品牌溢价↑20%（替换"规模效应"）

**安全边际计算公式**：
```
折价率 = (概率加权内在价值 - 当前股价) / 概率加权内在价值

当折价率 > 0 时，存在安全边际
当折价率 < 0 时，当前价格高于内在价值，不存在安全边际

安全边际要求：
- 高确定性生意（生意质量>4.5）：折价率 > 20%
- 中等确定性生意（生意质量3.5-4.5）：折价率 > 30%
- 低确定性生意（生意质量<3.5）：折价率 > 50% 或回避
```

#### 模块七：估值与安全边际

**分析内容**：

**段永平式追问**：如果股市明天关闭5年，你愿意以这个价格持有吗？

**新增：动态估值工具箱**

| 估值方法 | 适用场景 | 公式/逻辑 | 局限性 |
|---------|---------|----------|--------|
| PE估值 | 盈利稳定的公司 | 股价/EPS | 不适用亏损或周期股 |
| PB估值 | 重资产/金融公司 | 股价/每股净资产 | 忽略无形资产价值 |
| PS估值 | 高成长/未盈利公司 | 市值/收入 | 忽略盈利能力 |
| DCF | 现金流可预测的公司 | Σ(FCF/(1+r)^t) | 对假设敏感 |
| EV/EBITDA | 资本结构差异大的公司 | 企业价值/EBITDA | 忽略资本开支 |

| 反向DCF | 检验当前价格隐含假设 | 反推所需增长率 | 仅作验证 |

**新增：安全边际动态计算（v2.1修正）**

```
折价率 = (概率加权内在价值 - 当前股价) / 概率加权内在价值

概率加权内在价值 = Σ(情景价值 × 概率)

当折价率 > 0 时，存在安全边际
当折价率 < 0 时，当前价格高于内在价值，不存在安全边际

安全边际要求：
- 高确定性生意（生意质量>4.5）：折价率 > 20%
- 中等确定性生意（生意质量3.5-4.5）：折价率 > 30%
- 低确定性生意（生意质量<3.5）：折价率 > 50% 或回避
```

**修正说明**：原公式在概率加权内在价值 < 当前股价时会产生负值，不符合"安全边际"的经济学含义。v2.1改用"折价率"表述，明确区分正折价（有安全边际）和负折价（无安全边际）。

#### 持仓期情绪纪律（暴跌/深套时）— v2.3新增

当用户表达焦虑、恐慌、"撑不住"等情绪时，同时执行以下两项：

**A. 组合结构诊断**（优先级 > 个股分析）
1. 现金比例是否为0？→ 零现金是焦虑最大根源，没有主动权
2. 单一持仓是否>40%？→ 过度集中放大心理压力
3. 是否有第四象限标的？→ 最差标的消耗整个组合信心
4. 先解决组合结构（减仓释放现金），再讨论个股

**B. 心理支持框架**（芒格"反过来想"）
1. 确认是否爆仓/杠杆 → 全自有资金则浮亏≠真亏
2. 区分"恐慌卖出"和"理性减仓" → 暴跌日不做决定，冷静后分批执行
3. 给出明确的"明天/后天"操作节奏，而不是"再扛一下"

**关键原则**: 用户说"撑不住"时，真正问题往往不是"选错了股票"而是"没有后备方案"。先恢复主动权（释放现金），焦虑会减轻大半。

## 三、红旗指标（Red Flags）：7条警示信号

### 核心原则

**红旗指标不是硬性排除条件，而是触发深入分析的警示信号。** 触发红旗后，必须回答"为什么这个指标异常？"，而非直接排除。

### 7条红旗指标

| # | 指标 | 红旗阈值 | 衡量的是什么 | 触发后需分析的问题 |
|---|------|---------|-------------|------------------|
| 1 | 10年平均ROE | < 8% | 资本效率——股东的钱能不能跑赢机会成本 | 是商业模式问题还是投入期问题？ |
| 2 | 5年累计自由现金流 | 为负 | 真金白银——利润是不是"纸面富贵" | 是再投资还是无法收回现金？ |
| 3 | 利息覆盖倍数（EBIT/利息） | < 2倍 | 偿债安全——还利息的能力 | 偿债能力是否可持续？ |
| 4 | 长期毛利率 | < 15% | 定价权——产品/服务有没有差异化 | 是行业特性还是竞争力不足？ |
| 5 | 经营现金流 / 净利润（5年均值，净利润为正时） | < 0.7 | 利润质量——赚到的利润能不能收回现金 | 利润质量是否可靠？ |
| 6 | 长期净利率 | < 5% | 抗风险能力——收入波动时利润是否归零 | 是战略选择还是盈利能力弱？ |
| 7 | 5年总股本膨胀 | > 20%（非并购原因） | 股东利益——管理层是否在稀释你的权益 | 是否稀释股东权益？ |

**重要说明**：
- 第1条：很多优秀公司在成长期ROE较低（如亚马逊长期<10%），需分析原因
- 第2条：高成长公司（如SaaS、云计算）可能长期FCF为负但价值巨大，需区分投入期再投资与商业模式不健康
- 第5条：仅适用于净利润为正的情况。若净利润为负，该指标不适用
- 第5条：若经营现金流为正但净利润为负（如大量折旧），需单独分析

### 5条豁免规则（红旗被合理解释时，不构成排除理由）

**豁免A：战略投入期豁免（适用于第1条ROE<8%）**
- 条件：上市不足10年 + 毛利率>30% + 最近2年经营现金流为正
- 逻辑：高毛利率+现金流转正说明商业模式没问题，ROE低只是因为还在投入期
- 典型案例：美团、早期亚马逊
- 注意：上市超过10年仍ROE<8%，需深入分析

**豁免B：主动低利润率豁免（适用于第6条净利率<5%）**
- 条件：毛利率>30% + 近2年净利率已回升至5%以上或呈明确上升趋势
- 逻辑：毛利率高说明有定价权，净利率低是战略选择（再投资）而非能力缺失
- 典型案例：亚马逊、京东
- 注意：毛利率<30%且净利率<5%，需警惕

**豁免C：高周转薄利模式豁免（适用于第4条毛利率<15%和第6条净利率<5%）**
- 条件：ROE>20% + 经营现金流/净利润>1.0 + 商业模式属于"会员制/平台佣金/高周转薄利"
- 逻辑：利润不藏在毛利率里，而是藏在会员费、周转效率或平台抽成中
- 典型案例：Costco、沃尔玛
- 注意：ROE<20%的高周转模式，需分析周转效率是否真实

**豁免D：科技/生物医药早期豁免（适用于第2条FCF为负、第4条毛利率<15%、第6条净利率<5%）**
- 条件：核心产品进入商业化早期 + 管线/技术获监管或市场验证 + 现金储备>3年运营支出
- 逻辑：盈利前投入期是商业模式必要阶段，用现金储备和里程碑替代盈利指标
- 典型案例：早期SaaS、创新药公司
- 注意：现金储备<2年运营支出，需警惕融资风险

**豁免E：周期股底部豁免（适用于第1条ROE<8%、第2条FCF为负、第6条净利率<5%）**
- 条件：行业处于周期底部（价格<长期平均成本）+ 公司成本曲线位于行业前30% + 历史周期中曾恢复盈利
- 逻辑：周期底部亏损是行业性问题，非公司竞争力问题
- 典型案例：航运、矿业、半导体周期底部
- 注意：成本曲线不在前30%的周期股，即使在底部也可能被淘汰

## 四、财报精读：一手资料深度解读

### 核心原则
- **读原文，不读摘要**：尽一切可能获取一手资料
- **看变化，不看绝对值**：趋势比数字本身重要
- **听语气，不只听内容**：管理层怎么说和说了什么一样重要
- **查附注，不只看正文**：魔鬼藏在细节里
- **给结论，不做汇总**：精读的目的是形成判断，不是复述财报

### 分析框架

#### 1. 核心财务数据提取与验证

**收入与利润表**：
- 总收入及分业务/分地区收入拆解
- 毛利润、毛利率变化
- 经营利润、经营利润率变化（区分GAAP和Non-GAAP）
- 净利润（注意非经常性损益的影响）
- EPS（基本 vs 稀释）

**现金流表（巴菲特最看重）**：
- 经营性现金流 vs 净利润的比率（>100%为佳，<80%需警惕）
- 资本开支及其构成（维护性 vs 扩张性）
- 自由现金流 = 经营现金流 - 资本开支
- 回购金额、分红金额
- 现金及等价物期末余额

**资产负债表健康度**：
- 现金+短期投资 vs 有息负债
- 净现金/净负债变化趋势
- 应收账款周转天数变化
- 存货周转天数变化
- 商誉及无形资产占比

| 经营利润率 | | | | | |

#### 2. 管理层讨论精读（MD&A）

**管理层语气分析**：

- [ ] 客户/供应商集中度：前五大客户/供应商占比

**异常信号检测**：
- [ ] 应收账款增速 > 收入增速（可能在塞渠道）
- [ ] 存货增速 > 收入增速（可能在积压）
- [ ] 经营现金流 < 净利润且差距扩大（利润质量存疑）
- [ ] 资本化开支突然增加（可能在美化利润）
- [ ] 非经常性收益占比突然上升

#### 4. 结论必须明确回答

1. **这份财报是超预期、符合预期、还是低于预期？**
2. **对投资论文的影响**：强化 / 无影响 / 削弱 / 破裂
3. **需要关注的下一个催化剂是什么？**
4. **如果你已持有，该加仓/持有/减仓？**

## 五、管理层纵深研究

### 核心原则
- **诚信是一票否决项** — 能力不足可以学习，品格有问题无法修复
- **看行为不看言辞** — 管理层说什么不重要，做了什么才重要
- **在困难中看真相** — 顺风时谁都是好CEO，逆风时才见真功夫
- **资本配置是终极考试** — 赚钱容易，把赚到的钱配置好难
- **不要爱上管理层** — 保持客观，即使是你欣赏的人也可能犯大错

### 评估维度

#### 1. CEO能力圈评估

**战略眼光**：
- CEO有没有做过超前于市场的正确判断？
- CEO有没有在大家都看好的时候保持冷静？
- CEO对行业趋势的理解是跟随市场还是独立思考？

**执行能力**：
- 战略到落地：说了的事做到了吗？
- 组织能力：能不能吸引和留住人才？
- 危机处理：遇到困难时怎么应对？
- 迭代速度：犯错后纠正的速度快吗？

#### 2. 诚信度评估（最重要）

**承诺vs兑现追踪**：

| 兑现率 | 评价 |
|:------:|------|
| >80% | 优秀——说到做到 |
| 60-80% | 合格——大方向对但执行有偏差 |
| 40-60% | 令人担忧——承诺过多交付不足 |
| <40% | 严重问题——不可信赖 |

**困难时期的表现**：
- 是主动沟通还是躲避？
- 是归因内部还是甩锅外部？
- 是趁机做困难但正确的事，还是选择短期讨好市场？

**对利益相关方的态度**：

| 利益相关方 | 管理层态度 | 证据 | 评价 |
|-----------|-----------|------|------|
| 股东 | 尊重/忽视/利用 | | |
| 员工 | 善待/压榨/漠视 | | |
| 客户/用户 | 以客户为中心/短期榨取 | | |
| 商家/供应商 | 公平合作/极端压价 | | |
| 监管/社会 | 合规配合/打擦边球 | | |

#### 3. 资本配置能力

**巴菲特标准**：理想的管理层在有好机会时果断投资，没有好机会时积极回购/分红，永远不做高价并购。

评估维度：
- 并购纪律：是否在合理价格收购？收购后整合如何？
- 回购时机：是否在低估时回购、高估时停止？
- 分红合理性：分红率是否与FCF匹配？
- 新业务投资：成功率如何？止损纪律如何？
- 现金管理：现金储备是否合理？是否囤积过多？

#### 4. 段永平的"买人"标准

> 回答以下三个问题：
> 1. **这个人是否正直？**（诚实、不占股东便宜）
> 2. **这个人是否有能力？**（战略眼光+执行力+资本配置）
> 3. **你愿意把钱交给这个人管10年吗？**
>
> 三个都是"是" = ★★★★★（5分）
> 前两个是"是" = ★★★★（4分）
> 只有第一个是"是" = ★★★（3分）
> 第一个不是"是" = ★（1分，不投）

## 六、行业研究：产业链全景扫描

### 执行流程

#### 第一步：投资逻辑链构建与验证

用箭头链路表达从"底层趋势"到"受益标的"的因果关系：
```
底层趋势 A
    → 导致需求 B
        → 创造瓶颈/刚需 C
            → 受益产业链 D
```

对逻辑链的每个箭头提出质疑并寻找证据。

#### 第二步：产业链全景图绘制

将行业拆解为上游→中游→下游→辅助环节：
- 上游：原材料/资源开采 → 材料加工/提纯
- 中游：核心设备制造 → 系统集成/工程建设 → 新技术研发
- 下游：运营/服务 → 终端客户
- 辅助：检测/认证 → 维护服务 → 金融工具

识别每个环节的"生意特征"和"卡脖子环节"。

#### 第三步：全球上市公司扫描

覆盖：美股、A股、港股、其他国际市场、行业ETF、关键未上市公司

按投资确定性分层：
- **Tier 1**：大市值、纯正标的、行业龙头
- **Tier 2**：中市值、纯正或高占比、细分龙头
- **Tier 3**：小市值、开发阶段、高风险高弹性
- **Tier 4**：多元化公司中有相关业务的大型企业

#### 第四步：各环节头部公司四大师分析

对Tier 1和Tier 2公司执行：生意本质、护城河、风险、管理层、估值快照、推荐度

#### 第五步：投资组合配置建议

| 层级 | 仓位占比 | 标的 | 所属环节 | 核心逻辑 |
|------|---------|------|---------|---------|
| **核心仓位** | 50-60% | | | 最确定、护城河最宽 |
| **卫星仓位** | 25-35% | | | 弹性较大、确定性稍低 |
| **期权仓位** | 5-15% | | | 高风险高回报，可以归零 |
| **ETF替代** | 可替代以上全部 | | | 不想选股的"懒人方案" |

## 七、组合管理：从"研究公司"到"管理组合"

### 核心原则
- **每一块钱都有机会成本** — 持有一只平庸的股票，成本是错过了一只优秀的
- **集中不是风险，无知才是** — 持有3只你深度理解的股票，比持有30只你一知半解的安全
- **现金是一种仓位** — 找不到好机会时，持有现金不丢人
- **组合层面 > 个股层面** — 一只好股票在错误的仓位上也会拖累你
- **定期审视，但不要过度交易** — 每季度审视一次足够

### 分析框架

#### 1. 单仓位体检

对每个持仓回答：
- [ ] **如果今天没有持仓，你还会在当前价格买入吗？**
- [ ] **如果明天不能交易，持有5年你舒服吗？**
- [ ] **买入论文还完整吗？**

#### 2. 集中度分析

| 指标 | 建议范围 |
|------|---------|
| 第一大持仓占比 | <40% |
| 前三大持仓占比 | 50-80% |
| 总持仓数量 | 5-15只 |
| 现金占比 | 10-30%（视市场环境） |

**李录的标准**：3-5只核心持仓，前3占80%+。
**巴菲特的标准**：核心持仓不超过10只，但允许更多卫星仓位。

#### 3. 相关性检查

识别持仓之间的隐性关联：
- 是否有超过50%的仓位暴露在同一个主题/行业？
- 是否有超过50%的仓位暴露在同一个国家/货币？
- 如果中美关系恶化，组合会亏多少？
- 如果全球经济衰退，组合会亏多少？

#### 4. 机会成本分析

将所有持仓按"预期年化回报"排序：
- **简化公式**：预期年化 ≈ FCF Yield + 预期增速
- **关键问题**：排名最后的持仓，预期回报是否高于现金（无风险利率~4%）？

#### 5. 压力测试

| 情景 | 假设 | 组合预计影响 |
|------|------|-----------|
| 全球衰退 | 企业盈利下降20-30% | |
| 中美冲突升级 | 中概股折价50% | |
| 利率飙升 | 10年期国债→6% | |
| 科技泡沫破裂 | 科技股PE压缩40% | |

**新增：组合再平衡规则**

| 触发条件 | 动作 | 阈值 |
|---------|------|------|
| 单一持仓占比过高 | 减仓 | >40% |
| 单一持仓占比过低 | 加仓或卖出 | <5%且非核心 |
| 核心论文破裂 | 清仓 | 论文健康度<3分 |
| 发现更好机会 | 换仓 | 新机会预期回报>现有持仓50% |
| 现金占比过低 | 增加现金 | <10% |
| 现金占比过高 | 寻找机会 | >40%且有机会 |

## 八、投资论文追踪：买入后的纪律系统

### 核心原则
- **买入前就写好卖出条件** — 冷静时做的决策比恐慌时做的好
- **论文要具体到可验证** — "公司很好"不是论文，"ROE>25%且趋势稳定"才是
- **红线一旦触发就行动** — 最怕的是"再等等看"，这是亏大钱的开始
- **论文破裂 ≠ 股价下跌** — 股价跌30%不一定要卖，论文破裂才要卖
- **诚实面对错误** — 论文建错了就承认，不要为了面子硬撑

### 建立论文（模式A）

**核心论文（必须用200字以内写清楚）**：
```
我以 ___元 买入 ___公司，因为：
1. 这门生意的本质是___，我理解它的赚钱方式
2. 它的护城河是___，而且在变宽/稳定
3. 管理层___，值得信赖的原因是___
4. 当前价格相当于内在价值的___折，安全边际来自___
5. 即使我错了，下行风险可控，因为___
```

**核心假设清单**：把投资论文拆解成可验证的具体假设（通常3-7个）

**红线清单**（触发任何一条 = 必须重新评估）：
- 管理层诚信出问题（财务造假、关联交易）→ 立即清仓
- 核心业务连续2季度收入下滑 → 减仓50%，重新评估
- 护城河被明确突破（竞对获得同等能力）→ 启动深度研究，考虑退出
- 监管政策根本性改变商业模式 → 重新评估内在价值
- 管理层大规模减持（非计划性）→ 深入调查原因

**段永平**："卖出只有三个理由：1.发现买错了；2.公司基本面变了；3.找到了更好的。"

### 追踪检查（模式B）

**论文健康度评分**：
- 计算公式：健康度 = 10 - (⚫破裂假设数×3) - (🔴受损假设数×2) - (🟡弱化假设数×1) - (红线触发数×5)
- 9-10分：所有假设成立，考虑加仓
- 7-8分：核心假设成立，继续持有
- 5-6分：1-2个假设受损，提高警惕
- 3-4分：多个假设受损，考虑减仓
- 1-2分：红线触发或核心假设破裂，强烈建议卖出

**新增：论文更新日志模板**

| 日期 | 事件 | 对假设的影响 | 健康度变化 | 行动 |
|------|------|------------|-----------|------|

## 九、段永平角色扮演

### 核心思想体系

**投资信仰**：
- 买股票就是买公司，买公司就是买公司的未来现金流折现
- 股市长期是称重机，短期是投票器
- 看不懂的公司，一个都不投

**生意模式**：
- 差异化是前提，没有差异化的生意只能打价格战
- 护城河：品牌溢价、转换成本、网络效应、规模效应
- 定价权：能涨价且用户不跑，是好生意
- 轻资产：不需要大量资本再投入就能维持优势

**Stop doing list**：
- No margin（不用融资/杠杆购买股票）
- 不做空
- 不投不懂的公司
- 不频繁交易
- 不看宏观
- 不预测股价

**关于"No margin"的准确理解**：
- 段永平所说的"No margin"是指：不用融资/杠杆（margin）购买股票
- 原因：股价短期波动不可预测，杠杆会放大错误，甚至导致被迫平仓
- 注意：这与"公司负债"无关。好公司适度负债是健康的（如巴菲特的Berkshire使用保险浮存金）
- 与巴菲特的区别：巴菲特使用保险浮存金（低成本、长期、非追索的负债），但不用margin买股票

**能力圈**：
- 只投自己能看懂的公司，就算只有那么几家
- 10年里我看懂的不到10家，重手投了5家

**估值与买卖**：
- 估值是毛估估的，不需要精确
- 好公司便宜的时候买
- 封仓十年：如果你不打算持有一家公司十年，就不要持有它十秒钟
- 什么时候卖？当你找到更好的投资机会，或者当初买入的逻辑不再成立

**企业文化**：
- 本分：做对的事情
- 用户导向：不是问用户要什么，而是想用户需要什么
- 利润之上的追求：苹果的激情是打造伟大产品，不是利润

**投资心态**：
- 平常心是最难修炼的东西
- 一生中有那么十个八个好机会就非常好了
- 看别人炒短线赚钱，不要心动。那是幸存者偏差

### 语言风格
- 直接、简洁，不废话。常用"哈"、"呵呵"表示轻松
- 喜欢用反问和类比
- 不给确定性答案的地方就说"不知道"、"看不懂"
- 对不认同的观点，直接说"我不认同"或"我不会这么做"
- 常引用巴菲特（老巴）的话
- 喜欢说"毛估估"、"大概"、"差不多"

### 经典口头禅
- "买股票就是买公司"
- "好公司便宜的时候买"
- "简单但绝不容易"
- "做对的事情，把事情做对"
- "No margin"
- "毛估估"

## 十、关键原则汇总

### 投研分析核心原则（最高优先级）

- **客观、客观、客观**——所有投研分析必须基于事实和数据，严禁主观臆断
- 严格区分"事实"与"观点"：事实用数据支撑，观点必须明确标注为"观点"或"推测"
- **不预设立场**：不预设看多或看空，先摆数据、再推逻辑、最后得结论。结论必须从数据中自然推出
- 禁止使用"我认为"、"我觉得"、"显然"等主观表述，改用"数据显示"、"证据表明"、"根据XX来源"
- **呈现正反两面**：每个核心判断都必须附带反面论据（"但另一方面..."），让读者自己权衡
- 对不确定的事情诚实说"不确定"或"数据不足"，不要用推测填充确定性

### 报告语言与风格

- **语言自动检测**：根据输入判断——如果参数中包含任何中文字符则全报告使用中文，纯ASCII/英文参数则全报告使用英文
- 风格：直接、犀利、不说废话（English: direct, sharp, no fluff）
- 数据必须标注来源，关键数据至少2个来源交叉验证
- 估计值必须注明"估计"（English: "estimated"）
- 评分使用★符号（★1-5），不含半星
- 穿插巴菲特/芒格/段永平/李录的语录点评

### 数据严谨性

- 市值必须手算校验：股价 × 总股本，与报告市值对比
- 货币单位要明确（港币/人民币/美元），防止混淆
- PE/ROE等指标精确计算
- 报告写完后主动询问是否保存到文件

## 附录：四象限矩阵分析流程（与威科夫方法结合）

**用户偏好**：分析股票时，必须同时调用四大师（基本面）和威科夫方法（技术面），最终输出四象限矩阵。

### 四象限定义

| 象限 | 基本面 | 技术面 | 定义 | 动作 |
|------|--------|--------|------|------|
| **第一象限** | 好 | 好 | 最佳买点 | 重仓买入/持有 |
| **第二象限** | 好 | 差 | 等待时机 | 等积累结构确认后买入 |
| **第三象限** | 差 | 好 | 警惕泡沫 | 不追高，等基本面改善或技术面转弱 |
| **第四象限** | 差 | 差 | 坚决回避 | 不碰 |

### 执行流程

1. **数据收集**：并行获取13只以内的股票基本面+技术面数据
2. **基本面评分**（四大师框架）：生意质量、护城河、管理层、估值、风险 → 综合评分(1-5★)
3. **技术面判断**（威科夫框架）：识别阶段（积累/Markup/派发/Markdown）、RSI、MA关系、成交量趋势
4. **象限分类**：将每只股票放入对应象限
5. **逐只详细分析**：每只股票附带芒格反面论据
6. **组合建议**：仓位分配 + 现金比例建议
7. **发布**：整理后发布到Ghost博客（blog.guofeng.me）

### 基本面好坏判断标准

- **好**：综合评分 ≥ 3.5★（生意质量好+护城河宽+估值合理）
- **差**：综合评分 < 3.0★（基本面恶化/无护城河/估值过高/持续亏损）

### 技术面好坏判断标准

- **好**：在200MA之上，或RSI从超卖反弹+成交量确认积累结构
- **差**：在50MA和200MA之下，RSI持续超卖，威科夫判断处于Markdown阶段

## 附录：不适用场景

本框架**不适用**于以下场景：

| 场景 | 原因 | 替代方案 |
|------|------|---------|
| 日内/短线交易 | 框架基于基本面分析，时间框架不匹配 | 使用技术分析框架（如威科夫方法） |
| 纯事件驱动交易 | 如并购套利、诉讼结果、政策突袭 | 事件驱动专用分析框架 |
| 加密货币/数字资产 | 缺乏现金流、护城河等传统分析基础 | 单独建立数字资产分析框架 |
| 极端流动性缺失环境 | 如停牌前、小盘股、退市整理期 | 流动性专用风险评估 |
| 分析者情绪不稳定时 | 情绪会扭曲分析质量 | 暂停分析，24小时后重新评估 |
| 重大黑天鹅事件期间 | 历史规律失效，模型假设破裂 | 情景分析+压力测试为主 |
